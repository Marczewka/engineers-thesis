This folder holds extracted feature datasets used by the IRMAS notebooks.

Files:

- `melspectrograms/` — melspectrogram arrays (`X.npy`, `y.npy`)
- `mfcc/` — MFCC arrays (`X.npy`, `y.npy`)
- `mfcc_stats/` — MFCC statistics arrays (`X.npy`, `y.npy`)

These files are generated by the preprocessing notebooks in `notebooks/preprocessing/`. Do not commit large binary feature files; regenerate them with the notebooks when needed.
