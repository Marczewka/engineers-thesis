# Kaggle Metadata

This folder should contain the Kaggle competition metadata CSV files required by the preprocessing notebooks.

Place the metadata files extracted from `FSDKaggle2018.meta.zip` here before running `code/kaggle/notebooks/preprocessing/clean.ipynb` and `code/kaggle/notebooks/preprocessing/feature_extraction.ipynb`.

The preprocessing notebooks will generate extracted features in `code/kaggle/features/`.

Download the dataset from:

- https://www.kaggle.com/competitions/freesound-audio-tagging/data
